Message Translation Rules

A

Letters (A-Z)

Each letter lights up its corresponding LED position (A=LED 0, B=LED 1, etc.)

5

Numbers (1-26)

Numbers light up a sequence of LEDs (1=LED 0, 5=LEDs 0-4, etc.)

!

Special Characters

Any non-alphanumeric character clears all LEDs

Spaces

Spaces create a short pause between characters

Example: "HELLO 123!"

# CircuitPython will:
# H → Light LED 7
# E → Light LED 4
# L → Light LED 11 (twice)
# O → Light LED 14
# Space → Pause
# 1 → Light LED 0
# 2 → Light LEDs 0-1
# 3 → Light LEDs 0-2
# ! → Clear all LEDs
